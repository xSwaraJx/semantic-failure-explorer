Semantic Failure Explorer

Two AI agents coordinate conference tasks over an A2A-style protocol, each grounded in a different OWL ontology (CMT vs EKAW). Typed semantic mismatches are injected into their messages; every run below is a labelled interaction log. 106 runs · 85 induced failures · 25 silent-wrong outcomes. Pick a run to see the message exchange and verdict. — MSc project, Swaraj Gamare, University of Liverpool.
